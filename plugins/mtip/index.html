Installation (include this in the head section):

  • <link href='mtip.css' rel='stylesheet' />
  • <script src='jquery.js'></script>
  • <script src='mtip.js'></script>
  • <script>
$(document).ready(function(){
	$('.mtip').mtip();
});
</script>
Note: the selector can be anything you want, I just used the class mtip for the example
