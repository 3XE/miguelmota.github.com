Installation this goes in the head section

  • include jquery <script src='jquery.js'></script>
  • include mtip js <script src='mtip.js'></script>

  • include mtip stylesheet <link href='mtip.css' rel='stylesheet' />
      • initialize 
      		<script>
		$(document).ready(function(){
			$('.mtip').mtip();
		});
		</script>
      note: the selector can be anything you want, I just used the class mtip for the example
