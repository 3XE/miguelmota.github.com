    So how did I do this? First let's create an empty unordered list with a div below it that will serve as a loader:

    {% highlight html %}
      fetching bookmarks...
      {% endhighlight %}

      We are using the $.getJSON function. We're going to loop though the data with the $.each function and get the title, url and date. For the date we are going to create a new Date() instance and converted to a UTC string. Then we are appending a list item to our unordered list that we created earlier which will include our variables. So now after it complete the jquery, let's remove the loader:

      {% highlight javascript %} $.getJSON('http://feeds.delicious.com/v2/json/miguelmota?count=3&callback=?', function(data){ $.each(data, function(i, item){ var title = item.d; var url = item.u; var date = new Date(item.dt).toUTCString(); $('#delicious-bookmarks').append("
    • » "+title+" "+date+"
      • "); }); $('.loader').css('display','none'); } ); {% endhighlight %}

      But hold on, if we leave it like that, the date will show as a UTF string and will look like Tue, 11 Oct 2011 05:40:47 GMT. We want to display it as time ago so we will use James Padolsey's "pretty-date" function. Before the $.getJSON function, include:

      {% highlight javascript %} var niceTime = (function(){ var ints = { second: 1, minute: 60, hour: 3600, day: 86400, week: 604800, month: 2592000, year: 31536000 }; return function(time){ time = +new Date(time); var gap = ((+new Date()) - time) / 1000, amount, measure; for (var i in ints){ if (gap > ints[i]){ measure = i; } } amount = gap / ints[measure]; amount = gap > ints.day ? (Math.round(amount)) : Math.round(amount); amount += ' ' + measure + (amount > 1 ? 's' : '') + ' ago'; return amount; }; })(); {% endhighlight %}

      Now add the date variable as the niceTime() function argument like so:

      {% highlight javascript %} "+niceTime(date)+" {% endhighlight %}
